ยงMutations Utility Module

This module provides utility functions for creating common mutations to HTTP requests and responses in Envoy external processors. These functions simplify the creation of processing responses for modifying headers, bodies, and generating immediate responses.

ยงOverview

The mutations module includes functions for:

  • Adding and removing HTTP headers
  • Replacing HTTP body content
  • Clearing HTTP body content
  • Creating immediate responses with custom status codes
  • Creating redirect responses

These utilities abstract away the complexity of constructing the correct Envoy protobuf message structures, making it easier to implement common HTTP traffic modifications.
